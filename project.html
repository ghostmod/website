What is UrtDevs?

Welcome to UrTDevs! We are a community aimed at supporting the development of Urban Terror. Our main focus is to improve upon the game that we all love. There is active development in many projects related to Urban Terror, including custom clients, custom server binaries, pugbots, and irc bots. Click one of the project links to find out more!

diRf!

This client is in TESTING phase! Run at your own risk!

This client has not yet been approved for league use!

Features:

  • dmaHD
  • Includes network improvements from SACC.
  • Based on the latest ioq3 svn.
  • Ability to disable ambient noise: 
    s_loopingSoundsEnable (default 1)
 - Enable/disable environmental sounds in non-base maps.
		    
s_baseSoundsEnable (default 1)
 - Enable/disable environmental sounds in base maps. You must also set s_loopingSoundsEnable to 0 for this to work.
		    
s_localBreathingSoundsEnable (default 1)
 - Set cg_sfxBreathing 1 and disable this cvar to hear only enemies breathe but not yourself.
